Introduction

Coach2mqtt runs on a RaspberryPi 4B and allows you to control and monitor most of the systems in your Jayco Swift/Entegra Ethos or your Jayco Terrain/Entegra Launch from your phone or tablet.

On Renogy Lithium battery models you will be able to control most of the firefly devices (lights, fans, A/C, tank heaters, etc.), monitor the Renogy battery BMS (aka Samkoon screen), and also monitor and control your Renogy Inverter/Charger.

On generator models you will be able to control most of the firefly devices (lights, fans, A/C, tank heaters, etc.)

Lithionics models have not been tested.

About this App

Coach2mqtt runs as a collection of docker containers

Technology

In order to monitor and control the systems the RaspberryPi 4B connects to the RV-C bus and on Renogy Lithium battery models it also connects to the BMS CAN bus and the Inverter RS-485 port.

Getting Started

Hardware requirements

You will need the following:

  • RaspberryPi 4B with 2GB+ RAM
  • Waveshare 2-Channel CAN Hat+
    • Amazon
    Important: Be sure to buy the + version so you can supply power to the Pi and Hat from the RV-C cable.

  • A case for RaspberryPi 4B that can accomodate the CAN bus hat
    • Black HighPi Raspberry Pi Case for Pi 4 from Amazon
    • Black HighPi Raspberry Pi Case for Pi 4i from PiShop
    • White HighPi Raspberry Pi Case for Pi 4 from PiShop
    • Clear HighPi Raspberry Pi Case for Pi 4 from PiShop

  • Several feet of RV-C drop cable (4 conductor 24awg) or equivalant. Get enough so you have some flexibility where you mount the Raspberry Pi but keep in mind the RV-C spec says a drop cable should be no more than 6ft.
    • RV-C drop cable from FireFly integrations
    • 10ft with plugs attached already
    • 6ft with plugs attached already

  • One 3M miniplug connector to connect RV-C bus. Alternatively you can buy RV-C cable with a plug already attached (see above)
    • From Firefly Integrations
    • From Digikey

    • Renogy Lithium models also require these:

    • USB 2.0 A to B cable to connect to Renogy Inverter. Be sure to get one that is long enough to reach from the inverter to the where you are mounting the Pi.
      • Example from Amazon
    • Connector for Renogy BMS Debug port
      • 2 pin connector on Amazon

    Installing Coach2mqtt

    To fully utilize Coach2mqtt you will need to install an image on the Rasperry Pi and an app on your phone or tablet.

    Install the image

    1. Download and install the latest Imager tool

    2. Download the latest image for the Raspberry Pi

      Note: There is no need to decompress the downloaded image. The Imager tool can install a .zip image.

    3. Using the Imager tool install the image you downloaded onto the micro SD card
      1. Click CHOOSE OS <

      2. The app is installed now.

      Install the app on your phone or tablet
